# Analog Engineer Assistant

TUI for assisting analog electrical engineers. 

Provides a quick, helpful TUI guide with formulas for board and system level analog electrical design. 

Dual-license under MIT or the [UNLICENSE](https://unlicense.org/).

## Installation

Git clone the repo and in the directory run `cargo run --release` to see it in action. To jump to a section directly run `cargo run -- --adc`.
