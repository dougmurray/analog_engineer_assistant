# Formulas that need SolveVariants

- **The Basics**
    - Voltage Divider
        - solves_for: "R1"
            - expression: "R2 * V_in/V_out - R2"
        - solves_for: "R2"
            - epxression: "(V_out * R1) / (V_in - V_out)"
    - Inductor Impedance
        - solves_for: "L"
            - expression: "|Z_L| / (2 * pi * f)"
        - solves_for: "f"
            - expression: "|Z_L| / (2 * pi * L)"
    - Capacitor Impedance
        - solves_for: "C"
            - expresssion: "1 / (|Z_C| * 2 * pi * f)"
        - solves_for: "f"
            - expression: "1 / (|Z_C| * 2 * pi * C)"
    - LC Filter Corner Frequency
        - solves_for: "C"
            - expression: "1 / ((2 * pi * f)^2 * L)"
        - solves_for: "L"
            - expression: "1 / ((2 * pi * f)^2 * C"
    - RC Time Constant
        - solves_for: "R"
            - expression: "tau / C"
        - solves_for: "C"
            - expression: "tau / R"
    - Snubbing Capacitor
        - solves_for: "f_osc"
            - expression: "3 / (2 * pi * R_snub * C_snub)"
        - solves_for: "R_snub"
            - expression: "3 / (2 * pi * f_osc * C_snub)"
- **Amplifiers**
    - Non-Inverting Gain
        - solves_for: "R_f"
            - expression: "V_out * R_g/V_in - R_g"
        - solves_for: "R_g"
            - expression: "1/(V_out/(R_f * V_in) - 1/R_f)"
        - solves_for: "V_in"
            - expression: "-V_out/(-R_f/R_g - 1)"
    - Inverting Gain
        - solves_for: "R_f"
            - expression: "-V_out * R_g/V_in"
        - solves_for: "R_g"
            - expression: "-R_f * V_in/V_out"
        - solves_for: "V_in"
            - expression: "-V_out * R_g/R_f"
    - Noise Gain (Closed-Loop)
        - solves_for: "R_f"
            - epxression: "R_g * (G_n - 1)"
        - solves_for: "R_g"
            - expression: "R_f / (G_n - 1)"
    - Op-Amp Bandwidth
        - solves_for: "R_f"
            - epxression: "GBP * R_g/BW - R_g"
        - solves_for: "R_g"
            - expression: "1/(GBP/(R_f * BW) - 1/R_f)"
        - solves_for: "GBP"
            - expression: "BW * R_f/R_g + BW"
    - Small-Signal Rise Time
        - solves_for: "BW"
            - expression: "0.35 / t_rise"
    - Max Output Voltage (Slew Rate Limit)
        - solves_for: "SR"
            - expression: "2 * pi * f * V_peak"
        - solves_for: "f"
            - expression: "SR / (2 * pi * V_peak)"
    - Resistor Thermal Noise
        - solves_for: "R"
            - expression: "(V_n / 4 nV/sqrtHz)^2 * 1 kOhm"
    - In-Amp Common-Mode Filter Cutoff
        - solves_for: "R_in"
            - expression: "(1 / (2 * pi)) / (f_cm * C_cm)"
        - solves_for: "C_cm"
            - expression: "(1 / (2 * pi)) / (R_in * f_cm)"
- **ADC**
    - Full-Scale Range
        - solves_for: "V_REF"
            - expression: "PGA * FSR"
        - solves_for: "PGA"
            - expression: "V_REF / FSR"
    - LSB Size
        - solves_for: "FSR"
            - expression: "2^n * LSB"
